BBH-Sim

Reproducible Open-Source Binary Black Hole Simulation Framework

View on GitHub Get Started

Overview

BBH-Sim provides a reproducible pipeline for numerical-relativity simulations of binary black-hole (BBH) mergers. It defines standard parameter files, solver settings, and output formats for open-source reproducibility, enabling comparisons across codes such as the Einstein Toolkit, SpEC, BAM, and GRChombo.

Example Parameter File

A minimal, quasi-circular equal-mass BBH setup:

simulation_name: BBH_q1_chi0
units: geometric
G: 1
c: 1

initial_data:
  type: TwoPunctures
  m1: 0.5
  m2: 0.5
  spin1: [0.0, 0.0, 0.0]
  spin2: [0.0, 0.0, 0.0]
  separation: 12.0
  momentum1: [-0.05, 0.0, 0.0]
  momentum2: [0.05, 0.0, 0.0]

evolution:
  system: BSSN
  gauge:
    lapse: "1+log"
    shift: "Gamma-driver"
  grid:
    domain_radius: 1000.0
    amr_levels: 8
    dx_fine: 0.0125
  time:
    integrator: RK4
    t_final: 5000.0

Quick Start

  1. Clone the repository:
    git clone https://github.com/your-org/bbh-sim.git
  2. Build the Docker image:
    docker build -t bbh-sim docker/
  3. Run the toy example:
    bash examples/toy_lowres/run.sh
